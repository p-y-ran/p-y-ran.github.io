Privacy Policy

Last updated: March 10, 2026 | Effective date: March 10, 2026

This Privacy Policy is compliant with TikTok Developer Policies (2026), TikTok Global Data Processing Agreement (GDPA), Standard Contractual Clauses (SCCs) under GDPR, and China's Measures for Standard Contracts for Outbound Transfer of Personal Information.

This Privacy Policy describes how App0311 (operated by Foshan Hepburn Garment Industry Co., Ltd., "we", "us", or "our") collects, uses, stores, and protects your personal information when you use our application integrated with TikTok API (including video publishing functionality).

1. Information We Collect (Data Minimization)

We strictly adhere to TikTok's "data minimization" principle and only collect the following necessary public information:

  • TikTok Public Profile Data: Public nickname, avatar URL, user openID (for identity verification only), store order volume, unit price, transaction amount, inventory status (public data only), video views, likes, comments, conversion rate (public data only), competitor store data and best-selling product information (public data only)
  • Device Information (fully anonymized): No full IP address collected; only desensitized IP segments for security audit
  • Voluntarily provided information: Video files/links, captions, hashtags (only for authorized video publishing)

We DO NOT collect sensitive data (e.g., phone number, precise GPS location, biometrics, user private messages, payment information, competitor trade secrets, or non-public TikTok content).

2. How We Use Your Information

We use collected TikTok public data ONLY for the following TikTok-approved purposes:

  1. Analyze TikTok Shop operational data and video performance to generate reports for merchants.
  2. Provide market analysis based on public competitor data.
  3. Optimize content strategy by analyzing public video scripts.
  4. Authenticate users via TikTok Login Kit with minimum necessary permissions.
  5. Ensure service security and fraud prevention in compliance with TikTok rules.
  6. Upload and publish videos via official TikTok Content Posting API with user consent.

3. Data Compliance Commitment

  • No sensitive data collection.
  • All data transmitted via TLS 1.3 end-to-end encryption.
  • Data stored on Alibaba Cloud servers in mainland China.
  • Business data retained for 90 days; access logs for 30 days.
  • No data resale, no user profiling, no personalized advertising.

4. Data Sharing & Disclosure

We never sell, rent, or share your data except:

  • With TikTok to enable API functions.
  • As required by law or regulation.
  • To protect security and legitimate rights.

5. Children's Privacy

Our service is not intended for children under 13. We do not knowingly collect data from children under 13. If discovered, such data will be deleted within 24 hours.

6. Your Rights

  • Access your personal data
  • Correct inaccurate data
  • Request data deletion
  • Revoke authorization via TikTok App
  • Request data portability

7. Data Security

  • TLS 1.3 transmission encryption
  • AES-256 storage encryption
  • Role-based access control
  • Monthly security audits
  • Breach notification within 72 hours

8. Policy Changes

We may update this policy and notify users in advance when changes occur.
