Core Functions (TikTok Integration)

📊 TikTok Shop Data Analysis

Integrate with TikTok Shop API (sandbox mode) to access public shop operation data (order volume, sales, inventory) after user authorization, generating real-time reports for business optimization.

🎥 TikTok Authorized Video Publishing

With explicit user consent, assist in publishing videos to TikTok in compliance with TikTok content policies and rate limits, using official TikTok developer tools.

🔒 Data Compliance (TikTok Guidelines)

Strictly follow TikTok developer guidelines and data minimization principle: only collect public TikTok Shop data, no sensitive information, no data sharing, and all data can be revoked by users at any time.
